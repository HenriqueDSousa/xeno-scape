### Descrição

Neste jogo de plataforma em fases, inspirado na série Portal, você joga como um rato de laboratório, usado como cobaia para experimentos com portais. Seu objetivo é escapar do laboratório, aproveitando-se dos portais, sem ser pego pelos cientistas e robôs.

#### Mecânicas Básicas

O foco principal é a mecânica dos portais, implementada como uma Portal Gun capaz de gerar um portal azul e um laranja, sendo que o portal de uma cor teleporta para o outro objetos que fazem contato com ele. A velocidade é preservada durante o teleporte, possibilitando manobras interessantes e criativas. Também temos um tipo de bloco, os blocos pretos, que não permitem portais sobre eles, servindo como uma limitação para criar quebra-cabeças mais difíceis.

Uma outra mecânica importante é a de atirar, sendo usada tanto pelo jogador quanto pelos inimigos para causar dano. Por fim temos a mecânica de limite de tempo, que incentiva o jogador a pensar rápido e fazer melhor uso das outras mecânicas.

#### Condições de derrota e vitória

A condição de vitória de uma fase é atingir o bloco com o portal vermelho, que manda o jogador para a próxima fase, ou para o fim do jogo caso ele tenha vencido todas as fases.

As condições de derrota são: ser atingido por um inimigo, cair para fora da área de jogo, ou o tempo da fase acabar.
